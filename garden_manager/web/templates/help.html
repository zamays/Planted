üîß Troubleshooting

Can't find a plant?

Use the search bar or filter by season and type. You can also add custom plants if your desired plant isn't in the database.

Weather not loading?

The app works with mock weather data by default. To use real weather, add your OpenWeatherMap API key to the .env file (see SETUP.md).

How do I delete a plant from my plot?

Currently, you need to create a new plot to reorganize plants. A delete feature will be added in a future update.

Can I use this on mobile?

Yes! Planted is fully responsive and works great on phones and tablets. The interface adapts to your screen size.
